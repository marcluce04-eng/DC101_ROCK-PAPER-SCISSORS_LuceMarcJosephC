# DC101_ROCK-PAPER-SCISSORS_LuceMarcJosephC
Rock Paper A simple, universal hand game for 2+ players. On a shared signal, players form one of three shapes: rock (fist), paper (open hand), or scissors (two fingers). Rules: rock crushes scissors, scissors cut paper, paper covers rock. Ties are replayed. Used to make quick decisions or for casual fun, with no equipment required.
TECHNOLOGY USE:Visual Studio Code,Using html
